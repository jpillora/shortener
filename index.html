Subdomain Forwards

Subdomain Forwards allow you to use DNS to create short URLs.

Your domain is ready. You may create TXT records for each subdomain you wish to forward.

For example, to forward foo.example.com to http://my.example.com/long/url,
create a foo.example.com TXT record containing
subfwd-url=http://my.example.com/long/url
Please create a '*.{{ domain }}' (wildcard domain) CNAME record which points to 'handle.subfwd.com'
Incorrect CNAME '*.{{ domain }}' should be set to 'handle.subfwd.com'
